# 22.5 大语言模型（LLM）

> 出处：Kevin Murphy《Probabilistic Machine Learning: Advanced Topics》(MIT Press, 2023)，§22.5 Large Language Models (LLMs)，原书页码约 828–829。忠实翻译（信达雅）。

大语言模型（large language model, LLM）不过是一种对离散词元（token）建模的自回归生成模型，这些词元表示词语，也可能表示其他模态（例如图像块）的量化版本。此类模型几乎总是用变换器（transformer）实现，并采用极大似然进行训练。当下 LLM 是一个非常热门的课题，我们没有篇幅深入展开。不过，其基本思想已在第 22.4.1 节中总结，更多信息可参见诸如 [[XZ25](../reference.md#XZ25); [Bur25](../reference.md#Bur25); [Bro24](../reference.md#Bro24)] 等文献。
