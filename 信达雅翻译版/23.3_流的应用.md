# 23.3 流的应用

> 译自 Kevin P. Murphy,《Probabilistic Machine Learning: Advanced Topics》(MIT Press, 2023)，§23.3 Applications，原书页码约 846–848。忠实翻译（信达雅）。

在本节中，我们着重介绍流（flows）在若干规范化（canonical）概率机器学习任务中的应用。

## 23.3.1 密度估计

流模型允许精确计算密度，并可用于将多模态（multi-modal）密度拟合到观测数据上（参见图 23.3 的示例）。一个早期的例子是高斯化（Gaussianization）[[CG00](../reference.md#CG00)]，该工作将这一思想应用于低维密度的拟合。Tabak 与 Vanden-Eijnden [[TVE10](../reference.md#TVE10)] 以及 Tabak 与 Turner [[TT13](../reference.md#TT13)] 引入了流的现代理念（包括"标准化流"（normalizing flows）这一术语），将一个流描述为若干更简单映射的复合。深度密度模型（Deep density models）[[RA13](../reference.md#RA13)] 是最早使用神经网络作为流来参数化高维密度的工作之一。此后涌现了一系列丰富的后续工作，包括 NICE [[DKB15](../reference.md#DKB15)] 与 Real NVP [[DSDB17](../reference.md#DSDB17)]。（NVP 是 "non-volume-preserving"（非体积保持）的缩写，指变换的雅可比行列式（Jacobian）不为单位值这一事实。）掩码自回归流（masked autoregressive flows，第 23.2.4.2 节）进一步提升了在无条件与有条件密度估计任务上的性能。

流可用于构建混合模型（hybrid models），后者对输入与目标的联合密度 $p(x, y)$ 进行建模；与之相对，判别式分类模型只对条件分布 $p(y|x)$ 建模，而密度模型只对边缘分布 $p(x)$ 建模。Nalisnick 等人 [[Nal+19b](../reference.md#Nal+19b)] 提出了一种基于流的混合模型，利用可逆映射进行表示学习，并表明联合密度 $p(x, y)$ 可被高效计算，这对异常检测、半监督学习与选择性分类等下游任务很有用。基于流的混合模型在内存上是高效的，因为其大部分参数都位于可逆表示之中，而这些表示在判别式模型与生成式模型之间是共享的；此外，密度 $p(x, y)$ 可在单次前向传递中计算出来，从而带来计算上的节省。残差流（residual flows）[[Che+19](../reference.md#Che+19)] 使用可逆残差映射 [[Beh+19](../reference.md#Beh+19)] 进行混合建模，进一步提升了性能。流也曾被用于将密度拟合到嵌入（embeddings）[[Zha+20b](../reference.md#Zha+20b); [CZG20](../reference.md#CZG20)] 上，以用于异常检测任务。

## 23.3.2 生成建模

另一项任务是生成（generation），即从拟合得到的模型 $p^*(x)$ 中生成新颖的样本。生成是标准化流的一项流行下游任务，它已被应用于不同的数据模态，包括图像、视频、音频、文本，以及诸如图（graphs）与点云（point clouds）等结构化对象。图像可以说是深度生成模型最受欢迎的模态：GLOW [[KD18b](../reference.md#KD18b)] 是最早能够生成令人信服的高维图像的基于流的模型之一，并已被扩展到视频以生成 RGB 帧 [[Kum+19b](../reference.md#Kum+19b)]；残差流 [[Che+19](../reference.md#Che+19)] 也已被证明能够生成清晰锐利的图像。

Oord 等人 [[Oor+18](../reference.md#Oor+18)] 通过将 WaveNet 蒸馏（distilling）为一个 IAF（第 23.2.4.3 节）而将流用于音频合成，这使得采样比 WaveNet 更快。其他用于音频的流模型包括 WaveFLOW [[PVC19](../reference.md#PVC19)] 与 FlowWaveNet [[Kim+19](../reference.md#Kim+19)]，它们利用耦合层（coupling layers）直接对 WaveNet 进行加速。

流也曾被用于文本。Tran 等人 [[Tra+19](../reference.md#Tra+19)] 为语言建模任务定义了一种在词表（vocabulary）上的离散流（discrete flow）。另一种流行的方法是定义一个具有离散观测空间但连续潜在空间的潜变量模型。例如，Ziegler 与 Rush [[ZR19a](../reference.md#ZR19a)] 在潜在空间中使用标准化流来进行语言建模。

## 23.3.3 推断

标准化流已被用于概率推断。Rezende 与 Mohamed [[RM15](../reference.md#RM15)] 使标准化流在机器学习中得到普及，并展示了如何将其用于对潜变量模型中的变分后验分布（variational posterior distributions）进行建模。人们已提出多种扩展，例如 Householder 流 [[TW16](../reference.md#TW16)]、逆自回归流（inverse autoregressive flows）[[Kin+16](../reference.md#Kin+16)]、乘性标准化流（multiplicative normalizing flows）[[LW17](../reference.md#LW17)] 以及 Sylvester 流 [[Ber+18](../reference.md#Ber+18)]，用于对潜变量模型的变分后验进行建模，也用于贝叶斯神经网络（Bayesian neural networks）的后验。

流曾被用作重要性采样（importance sampling）的复杂提议分布（proposal distributions）；相关例子包括神经重要性采样（neural importance sampling）[Mül+19b] 与玻尔兹曼生成器（Boltzmann generators）[Noé+19]。Hoffman 等人 [[Hof+19](../reference.md#Hof+19)] 利用流来提升哈密顿蒙特卡洛（Hamiltonian Monte Carlo，第 12.5 节）的性能，其做法是定义双射变换（bijective transformations）将随机变量变换到更简单的分布，并改在该空间中执行 HMC。

最后，流可用于基于模拟的推断（simulation-based inference）这一情境中：此时参数的似然函数不可获得，但从模型中模拟数据是可行的。其主要思想是在模型所模拟出的数据上训练一个流，以逼近后验分布或似然函数。流模型还可用于引导模拟过程，从而使推断更加高效 [[PSM19](../reference.md#PSM19); [GNM19](../reference.md#GNM19)]。这一方法已被用于宇宙学 [[Als+19](../reference.md#Als+19)] 与计算神经科学 [[Gon+20](../reference.md#Gon+20)] 中模拟模型的推断。
