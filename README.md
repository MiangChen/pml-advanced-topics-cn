# PML Advanced Topics —— 中文精讲（乐高积木风格）

本仓库是 Kevin P. Murphy **《Probabilistic Machine Learning: Advanced Topics》** (MIT Press, 2023) 第四部分**「生成模型」(第 20–26 章)** 的**中文讲解笔记**：VAE、自回归模型、Normalizing Flow、能量模型、扩散模型、GAN。

> ⚠️ **这不是逐字翻译，而是「乐高积木」式精讲**——从最小的零术语直觉起步，一块一块往上扣，每块都看得见扣在哪，最后拼出原书那个"复杂大家伙"。**原书知识点 100% 不丢，公式 1:1 全搬。**

## 阅读方式

- 每个原书小节 `X.Y` 对应一个 `.md` 文件，按节号顺序读。
- 含 LaTeX 公式，建议用支持数学渲染的阅读器：**Typora / Obsidian / VS Code(Markdown+Math)**。
- 插图在 `figs/`（目前仅 20.1 配图；其余章节图表以文字讲解 + "原书 图X.Y" 标注呈现）。
