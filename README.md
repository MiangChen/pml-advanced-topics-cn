# PML Advanced Topics —— 中文精讲

本仓库是 Kevin P. Murphy 所著 **《Probabilistic Machine Learning: Advanced Topics》** (MIT Press, 2023) 的**中文讲解笔记**，目前覆盖**第四部分「生成模型」(第 20–26 章)**：VAE、自回归模型、Normalizing Flow、能量模型、扩散模型、GAN。

> ⚠️ **这不是逐字翻译，而是按小节重写的「精讲」**——先定义后展开、配直觉类比、对比成表、补原文潜台词，目标是让中文读者一遍读懂。

## 阅读方式

- 每个原书小节 `X.Y` 对应一个 `.md` 文件，按节号排序阅读。
- 含 LaTeX 公式，建议用支持数学渲染的阅读器：**Typora / Obsidian / VS Code(Markdown+Math)**。
- 插图在 `figs/` 目录。
